LOG LEVELS (6 Tiers)
─────────────────────────────────────────────────────────────────────────────
TRACE   (10)  │ Deep diagnostics, variable traces                   [Rarest]
DEBUG   (20)  │ Development info, debugging issues
INFO    (30)  │ General information, normal operations (DEFAULT)    [Default]
WARN    (40)  │ Warning conditions, something unexpected
ERROR   (50)  │ Error conditions, operation failed
FATAL   (60)  │ System failures, unrecoverable state               [Most Critical]

ENVIRONMENT SETUP
─────────────────────────────────────────────────────────────────────────────
Development:       export NODE_ENV=development LOG_LEVEL=debug
Production:        export NODE_ENV=production LOG_LEVEL=info
Staging:           export NODE_ENV=staging LOG_LEVEL=debug
Testing:           export NODE_ENV=test LOG_LEVEL=warn
Debugging:         export LOG_LEVEL=trace (max detail)

QUICK START
─────────────────────────────────────────────────────────────────────────────
1. Import:    import { createLogger } from '@agentic-sdlc/logger-config';
2. Create:    const logger = createLogger({ component: 'my-service' });
3. Use:       logger.info('message', { context });

LOG LEVEL DECISION TREE
─────────────────────────────────────────────────────────────────────────────
Is it user-visible output?           YES → INFO
Is it for development/debugging?     YES → DEBUG
Is it unexpected but handled?        YES → WARN
Did an operation fail?               YES → ERROR
Can system not continue?             YES → FATAL
Is it deepest detail?                YES → TRACE

COMMON LOGGING PATTERNS
─────────────────────────────────────────────────────────────────────────────
Info level (info):
  • Startup/shutdown messages
  • Major operations (workflow started)
  • Operation completion
  • Service initialization

Debug level (debug):
  • Function entry/exit
  • Variable values
  • Decision points
  • Loop iterations

Warn level (warn):
  • Deprecation warnings
  • Performance concerns
  • Recoverable errors
  • Temporary conditions

Error level (error):
  • Failed operations
  • Exceptions caught
  • Resource exhaustion
  • Connection failures

Fatal level (fatal):
  • System shutdown required
  • Unrecoverable state
  • Total service failure

PERFORMANCE TIPS
─────────────────────────────────────────────────────────────────────────────
✅ Use DEBUG (not INFO) for frequent messages
✅ Avoid string concatenation in objects
✅ Use DEBUG for loop iterations
✅ Lazy-evaluate expensive operations

❌ Don't log 1000x in a loop at INFO level
❌ Don't concatenate: logger.info('User ' + id + ' logged in')
❌ Don't log sensitive data (passwords, tokens)

COMMON MISTAKES
─────────────────────────────────────────────────────────────────────────────
❌ Using console.log (ignores LOG_LEVEL)
   Use: logger.info() instead

❌ String concatenation in logs
   Use: logger.info('msg', { var: value }) instead

❌ Logging secrets
   Use: logger.info('action', { userId }) instead

❌ Too much DEBUG logging
   Use: logger.debug('summary') instead of per-item logs

❌ Ignoring errors
   Use: logger.error() in catch blocks

GETTING STARTED (5 MINUTES)
─────────────────────────────────────────────────────────────────────────────
1. Set environment:   export LOG_LEVEL=debug NODE_ENV=development
2. Import logger:     import { createLogger } from '@agentic-sdlc/logger-config'
3. Create logger:     const logger = createLogger({ component: 'name' })
4. Use logger:        logger.info('msg', { context })
5. Run app:           npm start

─────────────────────────────────────────────────────────────────────────────

NODE_ENV=development LOG_LEVEL=debug npm start

# Production-like logging
NODE_ENV=production LOG_LEVEL=info npm start

# Debug specific issue
LOG_LEVEL=trace npm start

# Suppress logs
LOG_LEVEL=fatal npm start

# Check current setting
echo $LOG_LEVEL
